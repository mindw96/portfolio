주요 연구 프로젝트

1. License Plate Detection 성능 향상 연구

License Plate Detection과 License Plate OCR 2stage로 접근하였으며

Fast-SRGAN을 활용하여 License Plate의 해상도를 x4하는 아이디어 구상

전체 Accuracy 92.38%에서 96.72%로 향상

DOI: 10.30693/SMJ.2020.9.4.134
  • 역할: 아이디어 구상, 실험 설계, 실험, 논문 1저자 작성

2. OOD(Out Of Distribution) 분석 연구

모델의 관점에서 OOD 현상을 분석하고, 이를 통해 모델의 견고성을 향상시키는 연구를 수행

CIFAR10로 학습된 모델에 CIFAR100를 input으로 넣었을 때의 output값을 활용한 OOD 현상 시각화

  • 역할: 실험 설계, 실험

3. Texture Classification 연구

Texture Dataset에 수집한 데이터를 추가한 전체 데이터셋으로 VIT 모델을 학습

WebCam을 통해 실시간으로 촬영되는 물체의 Texture를 분류 및 시각화

  • 역할: 실험 설계, 실험, 데모 제작

4. 천체 데이터 프로젝트

  • Star, Galaxy, QSO 분류 연구
    • XGBoost 기반 분류 모델 개발
    • Feature Importance 분석을 통한 천체 데이터 특성 연구
  • 블랙홀 질량 예측 연구
    • XGBoost 기반 예측 모델 개발
    • 기존 딥러닝 모델 대비 향상된 성능 달성
    • 2년차 목표 조기 달성

5. LLM 기반 응용 프로젝트

  • SOLAR, EEVE, KULLM, BLLOSSOM 등을 4Bit Quantization Fine tuning을 통한 Eng-Ko 번역 모델 개발
  • SOLAR Embedding, Langchain을 활용한 IR 시스템 구축

6. OpenAI API를 활용한 웹앱

ChatGPT, DALL E API를 활용한 챗봇 웹앱 개발
